# oriskin-ekalosari

Landing page resmi untuk Oriskin Ekalokasari Bogor, dibangun dengan Astro.

## Tech Stack

- Astro 5
- Astro Sitemap
- Static assets di `public/`

## Menjalankan Proyek

```sh
npm install
npm run dev
```

## Build Production

```sh
npm run build
npm run preview
```

## Struktur Utama

```text
src/
	components/
	layouts/
	pages/
public/
```

## Deployment

- GitHub repository: https://github.com/madaldho/oriskin-ekalosari
- Vercel production URL: https://oriskin-ekalosari.vercel.app

## Catatan

Project ini berisi homepage klinik dengan section hero, tentang kami, layanan, produk, testimoni, FAQ, lokasi, dan CTA WhatsApp.
